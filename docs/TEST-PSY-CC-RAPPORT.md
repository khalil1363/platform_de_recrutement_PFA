# Test PSY CC — Rapports PDF & Excel

## Comment tester

1. Redemarrer le microservice recruitment (nouvelles colonnes + tables).
2. RH → **QCM** → **Nouveau QCM** → bouton **Charger modele PSY CC** → **Enregistrer**.
3. RH → **Candidats admis** → **Assigner QCM** (choisir le QCM PSY).
4. Candidat → login → **Mes evaluations** → passer le QCM (anti-triche actif).
5. RH → Details du candidat → **Rapport PDF** (Talent Insight style).
6. RH → **Export Excel suivi PSY** (meme structure que Suivi Test PSY CC).

## Structure rapport (inspiree de vos modeles)

### PDF
- Couverture + score global %%
- Introduction / precautions
- Talent Insight (barres de competences)
- Tableau competences (score / attendu / commentaire)
- Resume: points forts / points de developpement

### Excel
Colonnes: Nom & Prenom | Competences | Score / %% | Score attendu poste (%%) | Interpretation / Commentaire

18 dimensions: Approche client, Comprehension des besoins, Argumentation, Satisfaction client, Relationnel, Extraversion, Prise de decision, Prise de recul, Gestion du stress, Orientation qualite, Gestion des risques, Rigueur / regles, Initiative, Innovation, Planification, Leadership, Mentorat, Implication au travail.
